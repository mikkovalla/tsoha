# tsoha

tsoha-tietokantasovellus

--------------------------------------------------------------------------------

4. Html Suunnittelu: [Suunnitelmat](https://github.com/mikkovalla/tsoha/tree/master/app/views/suunnitelmat) [Views kansio](https://github.com/mikkovalla/tsoha/tree/master/app/views)
